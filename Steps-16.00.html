Add Title Logos and Canvas Background 16

1. Adding Project Title:
• Click on the ‘Text Box’ from the Home or Insert ribbon.
• Format and Align the Text Box: Set the height to 79, width to 1,278, horizontal to 1, and vertical to 0.
• Background and Color: Under Effects, Turn on the background and set the color to #FFFFFF with 0% transparency.
• Visual Border, Shadow, and Title: Turn these settings off.
• Type Project Title: Enter “Global Profitability Analysis by Categories (2012 - 2015)” in the text box.
• Format the Title: Double-click the title to adjust the font to Segoe UI, size 20, bold, center align, and font color #00498D.


2. Adding Colaberry and Superstore Logos:
• Open the ‘Insert’ tab from the ribbon and navigate to the ‘Elements’ section.
• Click on ‘Image’ and select the logo images from the extracted file.
3. Set the Colaberry Image Size and Style: Adjust the height to 72 and width to 205 with horizontal alignment at 1 and vertical alignment at 0.
4. Set the Superstore Image Size and Style: Adjust the height to 73 and width to 136 with horizontal alignment at 1,143 and vertical alignment at 0.
5. Apply Canvas Background
• Click blank space on the canvas
• On Format page in visualizations, click on Canvas background.
• Under Image, click Browse, and select the background image and click open.
• Set Image Fit = Fit, Transparency = 50%
• Wallpaper: Color #CCCCCC
• Transparency = 25%
