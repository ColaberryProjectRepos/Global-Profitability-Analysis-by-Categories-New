Add Sub-KPI Measure YOY% 21

1. Create the Initial Sub-KPI Measure YOY% Card:

  • Select the Visual Card: Drag it to the dashboard.
  • Drag the “Sales YOY%” measure: Drop it into the fields.
  • Data Label / Callout Value:
    • Font: Segoe UI Semibold
    • Font Size: 14
    • Category Label: OFF
  • General Properties:
    • Height: 35
    • Width: 68
    • Horizontal: 148
    • Vertical: 365
  • Color: #D9B300
  • Title Settings: OFF
  • Effects / Background: OFF

2. Duplicate and Modify the Remaining Sub-KPI Measure YOY% Cards:

  • GPM YoY% Card:

    • Duplicate the “Sales YOY%” Card.
    • Remove the “Sales YOY%” measure.
    • Replace it with the “GPM YoY%” measure.
    • Update General Properties:
      • Height: 35
      • Width: 68
      • Horizontal: 144
      • Vertical: 376
    • Set Color to #000000.

  • Discount YoY% Card:

    • Duplicate the “GPM YoY%” Card.
    • Remove the “GPM YoY%” measure.
    • Replace it with the Discount YoY% measure.
    • Update General Properties:
      • Height: 35
      • Width: 67
      • Horizontal: 147
      • Vertical: 487

  • Logistics YoY% Card:

    • Duplicate the “GPM YoY%” Card.
    • Remove the “GPM YoY%” measure.
    • Replace it with the Logistics YoY% measure.
    • Update General Properties:
      • Height: 35
      • Width: 67
      • Horizontal: 149
      • Vertical: 596
