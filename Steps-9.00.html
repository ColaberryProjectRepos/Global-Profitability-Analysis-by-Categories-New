Challenges and Solutions:

Overcoming data integration challenges and optimizing model complexity were crucial in enhancing the accuracy and applicability of predictive analytics.

1.

  1. Waterfall Chart with Multiple Measures: Initially, I thought Power BI’s waterfall chart only supported a single measure and a category column. Through research, I discovered it's possible to create a waterfall chart with multiple measures by using a variance measure to represent the differences between measures.
  2. Python Code Errors: I encountered significant errors and poor results with Python code due to a lack of theoretical understanding and familiarity with the model setup. Extensive research and debugging were required to diagnose and resolve these issues effectively.
  3. DAX Formula Development: Creating DAX formulas that accurately represented the data narrative was challenging, as some formulas failed to highlight key insights. I refined my approach by iterating on formulas and focusing on translating raw numbers into coherent stories for stakeholders.



Lessons Learned
The project underscored the importance of data integrity and preprocessing for successful predictive modeling. We recognized the value of iterative model testing for real-world applicability, refining our machine learning approach. These insights will guide future projects to align better with business objectives. Additionally, creating DAX formulas that accurately represented the data narrative was challenging, leading us to focus on translating raw numbers into coherent stories for stakeholders.
