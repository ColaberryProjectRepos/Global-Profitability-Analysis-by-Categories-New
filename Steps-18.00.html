Create KPI Cards 18
1. Add "Sales Per Month" KPI Card:
  • Open Visualization Pane > Select Visual Card >  Drag the visual to the dashboard 
  • Drag the Sales Per Month Measure into the fields.
  • General Settings:
  • Height: 111, Width: 205, Horizontal: 0, Vertical: 190
Data Label / Callout Value:
  • Set the Font type to DIN
  • Set the Font Size to 28
  • Set the Color to #00498D
  • Category: OFF
Title Settings:
  • ON, Text Title: "Sales Per Month"
  • Font Color: #FFFFFF, Background Color: ##17DADA
  • Font Size: 14, Font Family: Segoe UI Bold
  • Alignment: Center
Effects / Background:
  • ON, Color: #F2F1EF, Transparency: 0
