Add Country Map

1. Duplicate the "Sales Per Month" Card by using the copy and paste function

2. Select the Map Chart 

3. Remove fields where applicable and place the following measures and or columns into their appropriate fields

• Go to Table: Orders 

• Location: Country

• Bubble Size: Sales Per Month

 Tooltips:

• GPM%

• Sales YOY

• Logistics / Shipping

• Discount

Visual Formatting:

 Map Settings:

• Style: Road | Show Labels = ON

 Bubbles: 

• Range Scaling = AUTO | Size = -10

• Colors: 

• Default color = fx 

• Format Style = Gradient

• Based on field = Sales Per Month 

• Minimum = #D3062C

• Check "Add a Middle Color" | Center = #D3062C 

• Maximum = #0C9A5C

• Select Ok
